# Remocap

## ⚠️ GPT API Key 만료 안내

현재 사용 중인 **OpenAI(GPT) API Key**가 만료되어, 아래 기능들이 **일시적으로 작동하지 않습니다.**
- 감정 요약 기능  
- 챗봇 응답 생성

### ✅ 해결 방법

1. 새로운 OpenAI API Key 발급  
2. `local.properties` 파일에 환경 변수 추가
   ```
   OPENAI_API_KEY={키 값}
   ```
